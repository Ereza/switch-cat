Standard procedure for exporting:

1. nxdumptool: Navigate to all titles, dump Program romfs to sdcard.
2. Connect via USB File Transfer
3. Copy all files to temp directory
4. Run the following inside 0100000000000801 (error messages):
	find . -type f -name 'de*' -exec rm {} + && find . -type f -name 'fr*' -exec rm {} + && find . -type f -name 'es*' -exec rm {} + && find . -type f -name 'pl*' -exec rm {} + && find . -type f -name 'th*' -exec rm {} + && find . -type f -name 'it*' -exec rm {} + && find . -type f -name 'ja*' -exec rm {} + && find . -type f -name 'ko*' -exec rm {} + && find . -type f -name 'nl*' -exec rm {} + && find . -type f -name 'pt*' -exec rm {} + && find . -type f -name 'ru*' -exec rm {} + && find . -type f -name 'zh*' -exec rm {} + && find . -type f -name 'en-US*' -exec rm {} + && find . -type f -name 'common' -exec rm {} +
5. Rename en-GB to es:
	find . -iname "*" | rename -v "s/en-GB_/es_/g"
6. Copy all files manually (check for duplicates, normally the bigger file size is the good one):
	0100000000000801 to error_codes (remove DatabaseInfo)
	0100000000000804 to safety_html
	010000000000081D to error_display
	*.msbt files to msbt
	*.szs files to szs
7. Copy all szs files to the tools/sarctool/szs folder.
8. Run for each file:
	wine sarc_tool.exe -o <FILE.szs>, copy exported .msbt files to correct dir.
9. Export translations:
	switch-translation-tool export switchsystemmenu msbt/ nintendo-switch-menu-del-sistema-en_FWVERSION.csv
10. Export error texts:
	switch-error-tool export error_codes/ nintendo-switch-missatges-d-error-en_FWVERSION.csv
11. Upload to Weblate and translate as needed

Standard procedure for importing:

1. Remove source/target header from CSV, set line endings to LF only
2. Import translations:
	switch-translation-tool import switchsystemmenu msbt/ nintendo-switch-menu-del-sistema-ca.csv
3. Import error texts:
	switch-error-tool import error_codes/ nintendo-switch-missatges-d-error-ca.csv
4. Create SZS from MSBT one by one, using Uwizard.
5. Generate "contents" folder:
	./generate.sh EUes es
	./generate.sh EUfr fr
	./generate.sh EUit it
	./generate.sh EUen en-GB


IMPORTANT:

When any change occurs in common, error or starter, rebuild all languages by changing these strings:
	common.msbt.msbt/Cmn_Lang_EUes / EUfr / EUit / USen -> Català
	error.msbt.msbt/ErrCmn_LblErrCode_EUes / EUfr / EUit / EUen -> Codi d’error
	starter.msbt.msbt/Str_Decide_Spanish / French / Italian / English -> D’acord
